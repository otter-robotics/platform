📖 Объяснение концепций

Структура программы Arduino

Каждая программа состоит из двух обязательных функций. setup() выполняется один раз при включении. loop() повторяется снова и снова, пока плата включена.

void setup() { /* один раз */ } void loop() { /* бесконечно */ }

pinMode: настройка пина

Перед использованием пина нужно указать Arduino: это вход или выход? Для светодиода выход (OUTPUT).

pinMode(13, OUTPUT); // пин 13 = выход

digitalWrite: включить/выключить

HIGH подаёт напряжение на пин (светодиод горит), LOW убирает (светодиод гаснет).

digitalWrite(13, HIGH); // включить digitalWrite(13, LOW); // выключить

delay: пауза

Приостанавливает программу на указанное число миллисекунд. 1 секунда = 1000 миллисекунд.

delay(1000); // пауза 1 секунда
